image editing

Creating new images from old ones

Using two different images I edited them together to create one central image to describe the book "Dune" by Frank Herbert. This first project was a challenge for me, but little did I know all the other projects would get harder. I had never done anything with an editing or photoshop platform before, so using this was a whole new world for me. I had to crop, edit, and reshape images to make them fit the vision I had. I also changed the opaqueness of different images to change how they appeared on screen.
