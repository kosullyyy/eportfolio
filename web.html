Using HTML Javascript to create web pages I created a drag and drop quiz and a general knowledge quiz. HTML editing has been what I struggled with the most, as well as where I feel like I have had the most personal growth. Before this class I had 0 knowledge about how to edit these kinds of codes.

For the drag-n-drop quiz I first edited the HTML by changing the shuffle feature to true, the section titles to reflect my own content, and altered the answer choice options, as well as added options. For the CSS file I changed the background color, changed the list title color, and the size of the white background, and finally changed the size of the gray box around the answer choices.
